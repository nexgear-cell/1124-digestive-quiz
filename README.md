# Digestive System NCLEX Quiz (Static Web App)

This is a zero-backend quiz web app designed to be deployed on **Cloudflare Pages**.

## Files
- `index.html` – UI
- `app.js` – quiz logic
- `questions.json` – question bank

## Deploy to Cloudflare Pages (free)
1. Create a GitHub repo and upload these files.
2. In Cloudflare Dashboard → Pages → Create a project → Connect to Git.
3. Framework preset: **None** (static).
4. Build command: *(leave empty)*
5. Build output directory: `/` (root)

Cloudflare Pages product: https://pages.cloudflare.com/
